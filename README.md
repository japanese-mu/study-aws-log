# AWS Server Log

AWSとLinuxサーバーの学習ログです。  
EC2、VPC、Linux操作などを、実際に手を動かしながら少しずつ記録します。

## 目的

- AWSのサーバー周りを実務に近い形で学ぶ
- 学習内容をGitHub上に残し、後から見返せるようにする
- 詰まった点や失敗も含めて、思考の流れを記録する
- 将来的にQiita記事や転職用ポートフォリオへつなげる

## 学習方針

- 毎回、短くてもよいのでログを残す
- 完璧な理解よりも、まず触ることを優先する
- 1回ごとに「今日やったこと / 詰まったこと / 次にやること」を書く
- サーバー中心で進め、必要に応じてネットワークやAWS設計も学ぶ

## 今やっていること

- EC2の基本操作
- Linuxの基本コマンド
- Apacheなど簡単なWebサーバー構築
- VPCやセキュリティグループの基礎理解

## ログの書き方

各ログは以下の形で残します。

```md
# 2026-04-23

## 今日やったこと
- AWSコンソールにログイン
- EC2画面を確認
- インスタンス作成手順を確認

## 詰まったこと
- セキュリティグループの設定意図がまだ曖昧
- SSH接続の全体像がふわっとしている

## 次にやること
- EC2を1台作成する
- Apacheを入れてブラウザ表示を確認する
```

## メモ

- 学習ログなので、きれいに書くことより残すことを優先する
- 失敗や理解不足もそのまま残す
- あとでQiitaにまとめるときは、このログを元に整理する
