# VPC Basics

## VPCとは
AWS上でネットワークを分けるための仕組み。  
まずは「AWS内の自分用ネットワーク」くらいの理解から始める。

## 今の理解
- EC2を置くためのネットワークの土台
- 外部公開する/しない設計に関係する

## まだ曖昧なこと
- サブネットやルートテーブルと一緒に考えるもの
- パブリックサブネットとプライベートサブネットの違い
- インターネットゲートウェイの役割
- ルートテーブルの意味
- NAT Gatewayが必要な場面
